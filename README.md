# Simple Node Typescript Init #

### Usage ###
- Download package.json
- Download tsconfig.json

- Reconfig package.json project title (etc..)

- Structure project folder:
```
mkdir src
```

- Init it
```
npm install
```

- Start tsc watch
```
npm run dev
```

- Run app
```
npm run start
```
